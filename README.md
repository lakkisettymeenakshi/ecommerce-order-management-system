# E-Commerce Order Management System

Full-stack e-commerce order and inventory management application built using React.js, Spring Boot, and PostgreSQL.

## Tech Stack

Frontend

* React.js
* Axios
* Bootstrap
* JavaScript
* HTML/CSS

Backend

* Java
* Spring Boot
* Spring Data JPA
* Hibernate

Database

* PostgreSQL

## Features

* Add products
* View products
* Delete products
* Inventory management
* REST API integration

## Run Backend

```bash
cd backend
.\mvnw.cmd spring-boot:run
```

## Run Frontend

```bash
cd frontend
npm start
```

## API Endpoints

```text
GET /api/products
POST /api/products
DELETE /api/products/{id}
```
